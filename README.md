# Standing Together

A full-stack workspace and task management platform built with **React**, **Node.js**, **Express**, and **MongoDB**.

Users can create workspaces, invite members, manage projects and tasks, and collaborate — all with role-based access control and Google OAuth support.

Live: **https://basalt-tech.org**

---

## Tech Stack

**Backend**
- Node.js + Express.js (TypeScript)
- MongoDB + Mongoose v8
- Passport.js — JWT + Google OAuth 2.0
- Zod — request validation
- Jest + Supertest — 69 tests across 8 suites

**Frontend**
- React 18 + Vite (TypeScript)
- TanStack Query (React Query v5)
- Zustand — auth state
- React Hook Form + Zod
- shadcn/ui + Tailwind CSS
- react-i18next — English / French

**Infrastructure**
- GitHub Actions — CI/CD pipeline
- pm2 — process manager on VPS
- Nginx — reverse proxy

---

## Features

- Email + password registration and login
- Google OAuth 2.0 login
- JWT authentication (Bearer token, stored in sessionStorage)
- Create and manage workspaces
- Invite members via invite code
- Role-based permissions (Owner, Admin, Member)
- Create projects inside workspaces
- Create and assign tasks with status and priority
- Edit profile (name, profile picture)
- Delete account (cascades: deletes owned workspaces, projects, tasks, memberships)
- Reports and analytics per workspace
- Multilingual UI (English / French)

## Local Development

**Requirements:** Node.js 20+, MongoDB

**1. Clone**
```bash
git clone https://github.com/TarekzainAldin/Standing-together-2.git
cd Standing-together-2
```

**2. Backend setup**
```bash
cd backend
npm install
```

Create `backend/.env`:
```env
NODE_ENV=development
PORT=8000
BASE_PATH=/api

MONGO_URI=mongodb://localhost:27017/standing-together

JWT_SECRET=your_jwt_secret
JWT_EXPIRES_IN=1d

SESSION_SECRET=your_session_secret
SESSION_EXPIRES_IN=1d

GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_CLIENT_URI=http://localhost:8000
GOOGLE_CALLBACK_URL=http://localhost:8000/api/auth/google/callback

FRONTEND_ORIGIN=http://localhost:5173
FRONTEND_GOOGLE_CALLBACK_URL=http://localhost:5173/auth/google/callback
```

Seed roles (run once):
```bash
npm run seed
```

Start backend:
```bash
npm run dev
```

**3. Frontend setup**
```bash
cd frontend
npm install
```

Create `frontend/.env`:
```env
VITE_API_BASE_URL=http://localhost:8000/api
```

Start frontend:
```bash
npm run dev
```

Frontend runs at `http://localhost:5173`, backend at `http://localhost:8000`.

## CI/CD Pipeline

Every push to `master` triggers GitHub Actions:

```
push to master
      │
      ▼
1. Checkout + Setup Node 20
      │
      ▼
2. Backend: npm install → jest (69 tests) → tsc build
      │  any test failure blocks the deploy
      ▼
3. Frontend: npm install → vite build
      │  VITE_API_BASE_URL injected from GitHub Secret
      ▼
4. Delete node_modules (not copied to server)
      │
      ▼
5. SCP → copy all files to /var/www/Standing-together-2
      │
      ▼
6. SSH → npm install --production → pm2 restart
```

**GitHub Secrets required:**

| Secret | Description |
|--------|-------------|
| `VPS_HOST` | Server IP address |
| `VPS_USER` | SSH username |
| `VPS_SSH_KEY` | Private SSH key |
| `VITE_API_BASE_URL` | `https://basalt-tech.org/api` |
